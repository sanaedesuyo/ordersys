## 密码不匹配时

若使用上述账号登录失败，可能是数据库中的 BCrypt 哈希与当前运行环境不一致。可任选其一处理：

1. 重启后端（`DemoAccountInitializer` 会在启动时同步预置账号密码）
2. 执行迁移脚本：`docker exec -i ordersys-mysql mysql -uroot -proot123 ordersys < docker/mysql/migrate-auth.sql`
3. 在用户端注册一个新账号
